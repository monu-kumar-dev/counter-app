# Counter App

A simple Counter App built using HTML, CSS and JavaScript.

This project allows users to increase, decrease and reset the counter value. It also stores the counter value and action history using Local Storage, so the data remains available even after refreshing the page.

## Features

* Increase counter by 1
* Decrease counter by 1
* Increase counter by 5
* Decrease counter by 5
* Reset counter
* Save counter value using Local Storage
* Save action history
* Clear history option
* Dynamic color change based on counter value
* Simple and responsive UI

## Technologies Used

* HTML
* CSS
* JavaScript
* Local Storage

## How It Works

* The counter value is displayed on the screen.
* Users can increase or decrease the value using different buttons.
* Every action is stored in the history section.
* Counter data is saved in Local Storage.
* When the page is refreshed, the previous counter value and history are restored automatically.
* Users can clear the history using the Clear History button.
